La tiendita 

En este trabajo integrador se nos solicito hacer un inventario para una tienda en el cual se guarden, se editen, se eliminen y por supuesto
que se visualicen los productos.
La forma de funcionamineto es simple, como vista principal es la tabla de productos, en la parte superior de esta se encuentra el buscador 
donde se puede buscar por nombre por codigo o categoria, la busqueda se ira actualizando segun se ingresen caracteres en el buscador.
En la parte inferior de la tabla tenemos tres botones, cada uno tiene un clor diferente esto haciendo la aplicacion mas intuitiva:
color verde = en este boton hace un llamado hacia el metodo mostrarDialogoProducto en donde se llenaran los campos y en caso de llenar uno con datos erroneos o en blanco se pedira que se corriga
colo naranja = en este botos se edita el producto seleccionado, igualmente hace el llamado al metodo mostrarDialogoProducto solo que esta vez mostrara los datos del producto y se podra editar todo exeptuando el codigo 
color rojo = este boton hace la funcion de borrar el elemento seleecionado 
los datos se guardaran en un archivo txt que se crea a necesidad del programa
